# CamelUp

Play CamelUp on the terminal!

![](./media/UI.png)

## Requirements

- Ubuntu 20.04
- Ruby ~> 3.0

## Installation

```sh
$ bundle install
```

## Usage

### Play

```sh
$ bundle exec src/camelup.rb
```

### Raketask

- `bundle exec rake test` - run tests
- `bundle exec rake rubocop` - run rubocop checks
